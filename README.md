# 终信行
烬途遗章  
余烬散尽，天地成墟。你自焦土醒来，眸中只烙着失控时的灼光与嘶鸣，不知岁月几何。  
身下是龟裂的旧道，曾贯长街通衢，今唯见锈骨沉沙，裂痕如大地的泪痕。风卷尘沙，呼啸间皆作呜咽——这便是睁眼时的人间。  
执念，是独行的火种。日升月沉，你攥紧一封旧信，纸间字迹如温痕。它让你在力竭时再进一步，在绝境中夺一息喘息，携着这点跨越光阴的暖意，跋涉成孤碑。  
行囊中两瓶琉璃，是故人最后的馈赠。
而今荒道险障渐密，风中有铁锈与辐射的低语。你俯身疾驰，借信为盾，凭药为刃，在无序的废土上——  
与绝望竞逐，向未明之光奔去。

# 公众测试 1.3.x
【游戏模式调整】
1.进入中文测试阶段。
2.主要构建了新的实验模式（排位模式），原有的普通游戏模式改为回忆模式（练习模式），排位模式中技能冷却时间将会延长65%，排位结算分为败北、迷失、逃脱结局，败北基础将会扣除10点排位分（不同段位有不同的保护分），迷失不得分（不同段位有不同的保护分），逃脱结局有基础加分（不同段位有不同的加分），除外当连败超过三局后如果结局为逃脱/迷失有终止连败额外得分/侥幸逃脱得分。
3.支持在旅行日记中查看历史游戏记录，可以查看游戏结局、得分、段位变化（仅排位模式）、携带的技能。
4.支持在世界观中查看当前赛季和历史赛季的记录，包括存活率（胜率）、存活场数（逃脱场数）、实验场数（游戏场数）。
5.根据平衡性，删除了矫健天赋。
6.重新绘制遗信商店界面，添加购物车功能，方便购买信件（天赋或技能）。
7.重新绘制准备界面，并新增以下逻辑，当商店积分超过1200点时，必须一个及以上携带天赋才能进入游戏；增加天赋携带上限，每携带一个天赋需要30点天赋点数，可用总天赋点下调至90点。
【游戏内容调整】
1.受到伤害后，自动恢复1.0点生命值，恢复1.0点后，进入80秒冷却期，冷却期间不再触发该特质效果。
2.正在进行冷却的技能，在信件阶段更改后，将立即减免对应阶段的冷却比例。
3.玩家跳跃速度从原有70%上调至75%，信件阶段每升级一次，跳跃速度增加10%。
4.技能冷却时间减免比例优化，具体如下：一阶段减免15%、二阶段减免30%、三阶段减免45%。
5.优化希望信件各阶段解锁阈值，具体所需分数如下：一阶段150分、二阶段300分、三阶段550分、最终阶段600分。
6.若未达到逃脱分数，已解锁的信件将被销毁，玩家需重新购买相关信件。
7.玩家的移动速度随着阶段变化增加分别为一阶段4%、二阶段8%、三阶段12%。
8.“柯西序列”信件重做，现使用“柯西序列”可以抵御三次普通伤害，冷却时间为120秒。
9.障碍物生成间隔下调至1200ms，修复了亡灵生成位置错误的问题。
10.信件品质现分为四种稀有度级别，分别为“稀世”、“孤品”、“绝代”、“传奇”，信件可使用次数分别为1次/3次/5次/无限。
11.信件新增天赋“张狂”，“稀世”、“孤品”、“绝代”、“传奇”品质效果为，开局15/10/5秒后，可立即获得150点分数。
